Overview¶

This is a client for the Stackops Automation API. There’s a Python API (the automationclient module), and a command-line script (automation). Each implements 100% of the Stackops Automation API.

The code is hosted on Github

python-automationclient is licensed under the Apache License.

Automation¶

Through the python-automationclient you can get a fully and ready Openstack Installation from scratch using the Stackops ecosystem at all.

If you want to know more about what automation is, plesase go to Stackops Automation
